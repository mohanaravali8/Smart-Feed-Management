# 🐟 Smart Feed Management

A web-based application for sustainable aquaculture that helps users manage feed, monitor water quality, track environmental metrics, and maintain feed stock records.

## Features

- Feed Calculation
- Feed Consumption Trends
- Environmental Metrics
- Water Quality Analysis
- Species Gallery
- Fish & Shrimp Life Cycle
- Feed Stock Management
- Market Prices
- Community
- Feedback

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
